5. 查询多条数据为 map 集合

若查询的数据有多条时，并且要将每条数据转换为 map 集合

此时有两种解决方案

1、将 mapper 接口方法的返回值设置为泛型是 map 的 list 集合

结果：

{

password=123456,gender=男,id=1,age=23,email=1234@qq.com,username=admin

}

2、可以将每条数据转换的 map 集合放在一个大的 map 中，但是必须要通过 @MapKey 注解将查询的某个字段的值作为大的 map 的键

@MapKey(“id”)

结果：

{
1={password=123456, gender=男, id=1, age=23, email=12345@qq.com, username=admin},

​ 8={password=123456, gender=女, id=8, age=33, email=123@qq.com, username=root},

​ 9={password=123, id=9, username=admin}
